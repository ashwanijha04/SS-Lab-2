# SS-Lab-2
   

   
   ### Steps to compile and run Lex programs
   
   - **lex file_name.l**
   
   - **cc lex.yy.c -ll**
  
   - **./a.out**
   
   ### Steps to compile and run Yacc programs
   
   - **lex file_name.l**
   
   - **yacc -d file_name.y**
   
   - **cc lex.yy.c y.tab.c -ll -ly**
   
   - **./a.out**
