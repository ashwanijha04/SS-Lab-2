# SS-Lab-2
   **Before Starting you first need to install _flex_ and _bison_ packages** 

   
   ### Steps to compile and run Lex programs in terminal
   
   - **lex file_name.l**
   
   - **cc lex.yy.c -ll**
  
   - **./a.out**
   
   ### Steps to compile and run Yacc programs in terminal
   
   - **lex file_name.l**
   
   - **yacc -d file_name.y**
   
   - **cc lex.yy.c y.tab.c -ll -ly**
   
   - **./a.out**
