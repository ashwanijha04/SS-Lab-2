# SS-Lab-2
   

   
   ## Steps to compile and run Lex programs
   
   ### Step-1 
   **lex file_name.l**
   
   ### Step-2 
   **cc lex.yy.c -ll**
   
   ### Step-3 
   **./a.out**
   
   ## Steps to compile and run Yacc programs
   
   ### Step-1 
   **lex file_name.l**
   
   ### Step-2 
   **yacc -d file_name.y**
   
   ### Step-3 
   **cc lex.yy.c y.tab.c -ll -ly**
   
   ### Step-4 
   **./a.out**
